# Atul Portfolio

Premium, modern personal portfolio built with Next.js App Router, TypeScript, Tailwind CSS, and shadcn-style UI components.

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- Reusable shadcn-style UI primitives

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy on Vercel

1. Push this folder to a Git repository.
2. Import the repository in Vercel.
3. Vercel auto-detects Next.js from `vercel.json`.
4. Use the default build command: `npm run build`.
5. Deploy.

Node version is pinned using `.nvmrc` (`20`) for consistency.
